# LayerBatchRename 说明文档

## 脚本信息

| 项目   | 内容                           |
| ---- | ---------------------------- |
| 脚本名称 | 批量重命名图层                      |
| 文件名  | LayerBatchRename.jsx         |
| 版本   | 1.0.0                        |
| 适用版本 | Photoshop CS6 及以上（含 Beta 版本） |
| 功能   | 批量重命名选中图层，支持模式检测、变量替换和序号生成   |

***

## 使用方法

1. 在 Photoshop 中打开文档
2. 在图层面板中选中一个或多个图层/图层组
3. 执行脚本：`文件 > 脚本 > 浏览`，选择 `LayerBatchRename.jsx`
4. 在弹窗中编辑新表达式，实时预览新名称
5. 点击"应用"执行重命名但保持窗口打开，或点击"确定"执行并关闭
6. Ctrl+Z 可撤回上次重命名操作

- `重复使用可以将 LayerBatchRename.jsx 文件放到 [PS目录]\Presets\Scripts 目录下`
- `随后在 编辑-键盘快捷键-应用程序菜单-LayerBatchRename 中设置快捷键`

***

## 表达式语法

### 变量占位符 %1 ％2 ...

| 语法             | 原始表达式中的含义      | 新表达式中的含义          |
| -------------- | -------------- | ----------------- |
| `%1` `％2`  ... | 匹配任意文本（含空串和空格） | 引用原始表达式中对应位置的匹配文本 |

`%1` `％2`  ... 是自动检测生成的变量占位符，用于标记图层名中的变化部分。

在新表达式中，`%1`\~`%9` 引用原始表达式中捕获的文本，保留原始变化内容。

> **注意**：`%d` 为序号保留，不作为变量名使用。变量最多支持 61 个，超过时脚本会提示错误。

### 序号占位符 %d

| 语法   | 新表达式中的含义           |
| ---- | ------------------ |
| `%d` | 生成序号（位数由"位数"输入框控制） |

`%d` 在新表达式中用于生成序号。在新表达式中插入 `%d` 即可生成序号。

序号补零位数由"位数"输入框控制：位数 = 3 即补零至 3 位（如 001）。

### 特殊语法 %%

| 语法   | 含义          |
| ---- | ----------- |
| `%%` | 输出百分号字符 `%` |

若图层名中需要包含 `%` 字符，使用 `%%` 转义。

### 序号规则

序号从最下方选中图层开始，由"序号起始"值起，向上每层增加"序号步长"。

### 表达式自动检测算法

脚本使用递归算法自动分析图层名的模式：

1. **最长公共前缀**：提取所有图层名的共同前缀作为固定部分
2. **最长公共后缀**：提取剩余部分的共同后缀作为固定部分
3. **递归分析中间部分**：对前缀和后缀之间的变化部分递归应用相同策略
4. **最长公共子串回退**：当前缀和后缀都为空时，寻找所有名称中共有的最长子串，将名称分割后递归分析
5. **变量标记**：无法找到共同部分的区域标记为 `%1`、`%2`、`%3`...（按出现顺序编号）

该算法优先从字符串两端提取公共部分，能尽可能多地匹配重复部分，生成简洁直观的表达式。

### 使用示例

#### 复制图层场景

| 原始名称                     | 自动生成表达式 | 新表达式  | 起始 | 步长 | 新名称                   |
| ------------------------ | ------- | ----- | -- | -- | --------------------- |
| 图层 1, 图层 1 拷贝, 图层 1 拷贝 2 | 图层 1%1  | 图层 %d | 1  | 1  | 图层 1, 图层 2, 图层 3      |
| 图层 1, 图层 1 拷贝, 图层 1 拷贝 2 | 图层 1%1  | 新图层%1 | 1  | 1  | 新图层, 新图层 拷贝, 新图层 拷贝 2 |

> **提示**：复制图层时，`%1` 可匹配空串（如"图层 1"中 %1 匹配空），也可匹配含空格文本（如" 拷贝"、" 拷贝 2"）。将新表达式中的 `%1` 改为 `%d` 即可用序号替换整个变化部分。

#### 数字后缀场景

| 原始名称       | 自动生成表达式 | 新表达式  | 起始 | 步长 | 位数 | 新名称        |
| ---------- | ------- | ----- | -- | -- | -- | ---------- |
| 图层12, 图层22 | 图层%12   | 图层%13 | 1  | 1  | 0  | 图层13, 图层23 |
| 图层12, 图层22 | 图层%12   | 图层%d2 | 1  | 1  | 0  | 图层12, 图层22 |
| 图层12, 图层22 | 图层%12   | 图层%d  | 1  | 1  | 2  | 图层01, 图层02 |

> **提示**：`%1` 只占一个字符位置，后面的数字为字面量。表达式"图层%12"中 %1 是变量，"2"是固定文字。新表达式"图层%13"表示变量 %1 后接固定文字"3"。

#### 多变化部分场景

| 原始名称                 | 自动生成表达式     | 新表达式        | 起始 | 步长 | 新名称                  |
| -------------------- | ----------- | ----------- | -- | -- | -------------------- |
| 图层 1 文本 2, 图层 2 文本 3 | 图层 %1 文本 %2 | 图层 %d 文本 %d | 1  | 1  | 图层 1 文本 1, 图层 2 文本 2 |
| 图层 1 文本 2, 图层 2 文本 3 | 图层 %1 文本 %2 | 图层 %2 文本 %1 | 1  | 1  | 图层 2 文本 1, 图层 3 文本 2 |

#### 序号补零场景

| 原始名称                         | 原始表达式      | 新表达式       | 起始 | 步长 | 位数 | 新名称                          |
| ---------------------------- | ---------- | ---------- | -- | -- | -- | ---------------------------- |
| A\_1, A\_2, A\_3             | A\_%1      | A\_%d      | 0  | 5  | 3  | A\_010, A\_005, A\_000       |
| BG\_1\_X, BG\_2\_Y, BG\_3\_Z | BG\_%1\_%2 | FG\_%2\_%d | 1  | 1  | 0  | FG\_X\_1, FG\_Y\_2, FG\_Z\_3 |

***

| 区域      | 说明                                     |
| ------- | -------------------------------------- |
| 原始图层名列表 | 显示选中图层的当前名称，\[图]=普通图层，\[组]=图层组         |
| 新图层名列表  | 实时预览重命名后的名称，\[不匹配]=表达式无法匹配，\[空名称]=生成空串 |
| 原始表达式   | 自动检测的图层名模式，`%1`\~`%9` 匹配变量部分（含空串和空格）   |
| 新表达式    | 用于生成新名称，`%1`\~`%9` 引用原匹配文本，`%d` 生成序号   |
| 序号起始    | `%d` 序号的起始数值（可为负数，默认1）                 |
| 步长      | `%d` 序号每层递增的数值（可为负数，默认1）               |
| 位数      | 序号补零位数，0=不补零，3=补零为3位（如001），默认0         |
| 取消      | 关闭窗口，不执行任何操作                           |
| 应用      | 执行重命名但保持窗口打开，基于新命名重新生成表达式              |
| 确定      | 执行重命名并关闭窗口                             |

### 按钮行为说明

| 按钮 | 验证失败时             | 验证成功时                        |
| -- | ----------------- | ---------------------------- |
| 应用 | 提示错误，窗口保持打开，可继续修改 | 执行重命名，刷新窗口内容（原始名称和表达式更新为新命名） |
| 确定 | 提示错误，窗口保持打开，可继续修改 | 执行重命名，关闭窗口                   |

## 同名图层说明

Photoshop 允许不同图层使用相同名称。当新表达式生成的名称存在重复时：

- 重命名操作**正常执行**，不会被阻止
- 执行完成后弹窗**提示**存在同名图层，列出重复的名称
- 如需避免同名，可在提示后撤销（Ctrl+Z）并修改表达式

***

## 注意事项

1. **图层名中的 %**：若图层名需要包含 `%` 字符，在新表达式中使用 `%%` 转义
2. **同名图层**：Photoshop 允许同名图层存在，脚本会在执行后提示同名情况
3. **撤销**：每次点击"应用"或"确定"执行的重命名合并为一条历史记录，Ctrl+Z 一次撤回该次操作
4. **图层组**：图层组与普通图层使用相同的重命名逻辑，列表中以 \[图层组] 标识
5. **%1\~%9 与 %d 的区别**：`%1`\~`%9` 在原始表达式中匹配任意文本（含空串），在新表达式中引用捕获文本；`%d` 仅在新表达式中生成序号
6. **变量只占一个字符**：`%1` 后的数字是字面量，如"图层%12"表示"图层" + 变量1 + "2"
7. **应用按钮**：点击应用后，原始图层名和表达式会更新为重命名后的结果，可继续编辑进行多轮重命名
