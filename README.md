# LayerBatchRename 说明文档

在Photoshop中多选图层进行批量重命名的工具
自动生成图层名称表达式、保留差异内容和图层序号生成

## 使用方法

1. 下载项目的zip包并解压
2. 在 Photoshop 中打开文档
3. 在图层面板中选中多个图层/图层组
4. 执行脚本：`文件 > 脚本 > 浏览`，选择 `LayerBatchRename.jsx`
5. 在弹窗中编辑新表达式，实时预览新名称
6. 点击"应用"执行重命名但保持窗口打开，或点击"确定"执行并关闭
7. Ctrl+Z 可撤回上次重命名操作

- 需要重复使用时，建议将 `LayerBatchRename.jsx` 文件放到 `[PS目录]\Presets\Scripts` 目录下
- 随后可在 `编辑-键盘快捷键-应用程序菜单-LayerBatchRename` 中设置快捷键

***

## 表达式语法

以下内容会根据图层名称自动生成，只需对`新表达式`进行修改即可，点击重置可恢复默认表达式。

### 变量占位符 %1 ％2 ...

对比图层名中存在差异的内容进行占位，以保留图层命名中的差异部分

`%1` `％2`  ... 是自动检测生成的变量占位符，用于标记图层名中的变化部分。

在新表达式中，`%1`\~`%9` 引用原始表达式中捕获的文本，保留原始变化内容。

> **注意**：`%d` 为序号保留，不作为变量名使用。变量最多支持 61 个，超过时脚本会提示错误。

### 序号占位符 %d

在新表达式中插入 `%d` 即可生成序号。

"位数"控制序号补零：位数 = 3 即补零至 3 位（如 001）。

<img width="580" height="565" alt="image" src="https://github.com/user-attachments/assets/9e017c76-b4c0-4c41-8fee-282c1e7a31dd" />

### 序号规则

序号从最下方选中图层开始，由"序号起始"值起，向上每层增加"序号步长"。勾选"倒序"后序号从最上方图层开始递增。

### 使用示例

| 原始名称                        | 自动生成表达式  | 新表达式     | 起始 | 步长 | 新名称                     |
| --------------------------- | -------- | -------- | -- | -- | ----------------------- |
| 闪光,闪光 1, 闪光 1 拷贝, 闪光 1 拷贝 2 | 图层%1     | 图层-%d    | 1  | 1  | 闪光-1,闪光-2, 闪光-3, 闪光-4 |
| 角色-笑-眼睛, 角色-笑-嘴, 角色-哭-眼睛    | 角色-%1-％2 | 角色-%2-％1 | 1  | 1  | 角色-眼睛-笑 角色-嘴-笑, 角色-眼睛-哭 |

> **提示**：复制图层时，`%1` 可匹配空串（如"闪光 1"中 %1 匹配空）

| 原始名称                  | 自动生成表达式 | 新表达式    | 起始 | 步长 | 位数 | 新名称                   |
| --------------------- | ------- | ------- | -- | -- | -- | --------------------- |
| 图层1002, 图层1005,图层1007 | 图层100%1 | 图层100%d | 1  | 1  | 0  | 图层1001, 图层1002,图层1003 |
| 图层102, 图层202          | 图层%102  | 图层%d    | 1  | 1  | 3  | 图层001, 图层002          |

> **提示**：`%1` 只占一个字符位置，后面的数字为字面量。表达式"图层%12"中 %1 是变量，"2"是固定文字。新表达式"图层%13"表示变量 %1 后接固定文字"3"。

## 其他功能说明

| 区域     | 说明                       |
| ------ | ------------------------ |
| 排除图层组  | 开启后不对图层组进行重命名            |
| 排除图层   | 开启后不对普通图层进行重命名           |
| 获取子对象  | 开启后将选中图层组的所有子对象（含嵌套）加入列表 |
| GitHub | 复制项目 GitHub 链接到剪贴板       |

## 注意事项

1. **图层名中的 %**：若图层名需要包含 `%` 字符，在新表达式中使用 `%%` 转义
2. Photoshop 允许不同图层使用相同名称。当新表达式生成的名称存在重复时会弹窗**提示**存在同名图层，列出重复的名称，如需避免同名，可在提示后撤销（Ctrl+Z）并修改表达式
